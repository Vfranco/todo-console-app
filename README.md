## Pasos para ejecutar la aplicación

### Clonar aplicación

```bash
    git clone https://github.com/Vfranco/todo-console-app.git
```

### Ingresar a la aplicación
```bash
    cd todo-console-app
```

### Ejecutar npm install para la instalación de las dependencias
```bash
    npm install
```

### Levantar Aplicación
```bash
    npm run start
```
