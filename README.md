## Pasos para ejecutar la aplicación

### Clonar aplicación

```bash
    git clone https://github.com/Vfranco/todo-console-app.git
```

### Ingresar a la aplicación
```bash
    cd todo-console-app
```

### Instalar dependencias
```bash
    npm install
```

### Compilar Aplicación
```bash
    npm run build
```

### Levantar Aplicación
```bash
    npm run start
```
